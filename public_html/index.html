Summary
The art that Bishop, and I chose to recreate was really simple. Just a penrose triangle,
we were orignally hoping to create a program that would create penrose polygons, but that
seemed to be a little out of reach in the time we had. So in the end we decided on
something a little simpler.

Our interpretation is really simple, really all we did was flip it upside down and
due to our inability to color the inside, we opted to change the color of the lines.
we chose more saturated colors because the pastel colors were too faint.

The code is really simple, it just draws line between coordinates that we input.
